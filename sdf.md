## Overview

A bilingual personal portfolio site built from scratch using Django,
designed to showcase projects, professional experience, and skills
in both English and Persian.

## Tech Stack

- **Backend:** Django, django-parler for translations
- **Templating:** Jinja2 via django-jinja
- **Styling:** Tailwind CSS v4 with DaisyUI
- **Media:** Cloudinary for image and file storage
- **Dependency Management:** Poetry

## Key Features

- Full English/Persian translation with RTL support
- Dark/light theme toggle with persistent preference
- Markdown-powered project and resume content
- Responsive design across all screen sizes
